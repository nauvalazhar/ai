Install the kit in a React Router v7 project.

## Create project

Scaffold a React Router app. The v7 template ships with Tailwind and a `~/*` alias for `./app/*`.

<CodeTabs>
```bash title="bun"
bun create react-router@latest
```

```bash title="npm"
npx create-react-router@latest
```

```bash title="pnpm"
pnpm create react-router@latest
```

```bash title="yarn"
yarn create react-router
```
</CodeTabs>

If your template skips Tailwind, install it before running init. The CLI exits if Tailwind isn't installed.

## Initialize

Run init to install base deps, append the token CSS, and create the `cn` helper.

<CodeTabs>
```bash title="bun"
bunx @nauvalazhar/ai init
```

```bash title="npm"
npx @nauvalazhar/ai init
```

```bash title="pnpm"
pnpm dlx @nauvalazhar/ai init
```

```bash title="yarn"
yarn dlx @nauvalazhar/ai init
```
</CodeTabs>

The CLI prompts to confirm before touching any files. Pass `-y` to skip.

## What init does

- `@base-ui/react`, `class-variance-authority`, `clsx`, `tailwind-merge` added to `package.json`
- Token CSS appended to `app/app.css`
- `cn` helper created at `app/lib/utils.ts`
- `aikit.json` written with `paths.components: "app/components/ai"` and `imports: { "components/ai": "~/components/ai", "lib": "~/lib" }`

React Router projects use `~/*` as the alias for `./app/*`. Confirm it's wired in `tsconfig.json`:

```json title="tsconfig.json"
{
  "compilerOptions": {
    "paths": {
      "~/*": ["./app/*"]
    }
  }
}
```

## Add a component

<CodeTabs>
```bash title="bun"
bunx @nauvalazhar/ai add message
```

```bash title="npm"
npx @nauvalazhar/ai add message
```

```bash title="pnpm"
pnpm dlx @nauvalazhar/ai add message
```

```bash title="yarn"
yarn dlx @nauvalazhar/ai add message
```
</CodeTabs>

The file lands at `app/components/ai/message.tsx` with imports rewritten to `~/`. Use it from any route:

```tsx title="app/routes/_index.tsx"
import { Message, MessageAvatar, MessageContent } from "~/components/ai/message";

export default function Index() {
  return (
    <Message>
      <MessageAvatar />
      <MessageContent>Hi there.</MessageContent>
    </Message>
  );
}
```

## Dark mode

Set `.dark` on `<html>` in `app/root.tsx`, either statically or from a theme loader:

```tsx title="app/root.tsx"
export default function App() {
  return (
    <html lang="en" className="dark">
      <head>{/* ... */}</head>
      <body>{/* ... */}</body>
    </html>
  );
}
```

For runtime switching, wire a session-backed theme value or use a library like `remix-themes`.

## Font family (recommended)

The kit was designed around Inter and JetBrains Mono. Add the following to `app/app.css`:

```css title="app/app.css"
@theme inline {
  --font-sans:
    "Inter", ui-sans-serif, system-ui, sans-serif, "Apple Color Emoji",
    "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji";
  --font-mono:
    "JetBrains Mono", ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas,
    "Liberation Mono", "Courier New", monospace;
}
```
