Install the kit in a Next.js project (App Router).

## Create project

Scaffold a Next.js project. The `--yes` flag accepts the defaults, which include TypeScript, Tailwind, App Router, and the `@/*` alias.

<CodeTabs>
```bash title="bun"
bun create next-app --yes
```

```bash title="npm"
npx create-next-app@latest --yes
```

```bash title="pnpm"
pnpm create next-app --yes
```

```bash title="yarn"
yarn create next-app --yes
```

</CodeTabs>

If you opted out of Tailwind during setup, add it before running init. The CLI exits if Tailwind isn't installed.

## Initialize

Run init to install base deps, append the token CSS, and create the `cn` helper.

<CodeTabs>
```bash title="bun"
bunx @nauvalazhar/ai@latest init
```

```bash title="npm"
npx @nauvalazhar/ai@latest init
```

```bash title="pnpm"
pnpm dlx @nauvalazhar/ai@latest init
```

```bash title="yarn"
yarn dlx @nauvalazhar/ai@latest init
```

</CodeTabs>

The CLI prompts to confirm before touching any files. Pass `-y` to skip.

## Add a component

<CodeTabs>
```bash title="bun"
bunx @nauvalazhar/ai@latest add message
```

```bash title="npm"
npx @nauvalazhar/ai@latest add message
```

```bash title="pnpm"
pnpm dlx @nauvalazhar/ai@latest add message
```

```bash title="yarn"
yarn dlx @nauvalazhar/ai@latest add message
```

</CodeTabs>

The file lands at `components/ai/message.tsx` with imports rewritten to your alias. Use it from a server or client component:

```tsx title="app/page.tsx"
import { Message, MessageContent, MessageText } from "@/components/ai/message";

export default function Page() {
  return (
    <div className="max-w-lg mx-auto">
      <Message>
        <MessageContent>
          <MessageText variant="bubble">Hi, there!</MessageText>
        </MessageContent>
      </Message>
    </div>
  );
}
```

## Dark mode

`next-themes` is the simplest way. Wrap your `app/layout.tsx` with the provider and set the `class` attribute on `<html>`:

```tsx title="app/layout.tsx"
import { ThemeProvider } from "next-themes";

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body>
        <ThemeProvider attribute="class" defaultTheme="system">
          {children}
        </ThemeProvider>
      </body>
    </html>
  );
}
```

The kit picks up `.dark` from any ancestor.

## Font family (recommended)

The kit was designed around Inter and JetBrains Mono. Add the following to `app/globals.css`:

```css title="app/globals.css"
@theme inline {
  --font-sans:
    "Inter", ui-sans-serif, system-ui, sans-serif, "Apple Color Emoji",
    "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji";
  --font-mono:
    "JetBrains Mono", ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas,
    "Liberation Mono", "Courier New", monospace;
}
```
