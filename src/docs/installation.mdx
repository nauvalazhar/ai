Pick the install path for your framework.

## Frameworks

- **[Vite](/installation/vite)** — token CSS appended to `src/index.css`, `@/` alias for `./src/*`.
- **[Next.js](/installation/next)** — token CSS appended to `app/globals.css`, `@/` alias for the project root.
- **[React Router](/installation/react-router)** — token CSS appended to `app/app.css`, `~/` alias for `./app/*`.
- **[TanStack Start](/installation/tanstack-start)** — token CSS appended to `src/styles.css`, `@/` alias for `./src/*`.

## Quickstart

If you're on one of the four frameworks above, the install is two commands:

<CodeTabs>
```bash title="bun"
bunx @nauvalazhar/ai@latest init
bunx @nauvalazhar/ai@latest add message
```

```bash title="npm"
npx @nauvalazhar/ai@latest init
npx @nauvalazhar/ai@latest add message
```

```bash title="pnpm"
pnpm dlx @nauvalazhar/ai@latest init
pnpm dlx @nauvalazhar/ai@latest add message
```

```bash title="yarn"
yarn dlx @nauvalazhar/ai@latest init
yarn dlx @nauvalazhar/ai@latest add message
```

</CodeTabs>

`init` writes `aikit.json` and sets up tokens, deps, and the `cn` helper. `add <name>` copies a component into your project with imports rewritten to your alias. Run `add` for each component you want.
